### Languages :
- Java 
- Xml
### Android components:
 - Activity
 - Fragment
 ### Libraries 
 - Retrofit 2.9.0 (https://square.github.io/retrofit/)
 - Picasso 2.71828 (https://square.github.io/picasso/)
